Programmer Documentation

This documentation explains to programmers how to modify the data.json file
to add new data / a new cat type. All of the information of cat types are in data.json.

How to add a new cat

Add the following inside the "breeds" bracket:

    {
        "name": "new_breed_name",
        "image": "images/new_img_name.jpg",
        "description": "New breed description."
    },
